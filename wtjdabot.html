Creation

A chat bridge is a popular tool for Minecraft servers, it allows users from Discord to talk to players in-game. A chat bridge on Minecraft is no easy feat, but unlike War Thunder, Minecraft has APIs and plugins. I had to be the first to figure out how to interact with the game, how to get data from its undocumented ports, how to write to it. The discord side of things was relatively easy, as JDA (Java Discord Api) is well documented and used, but along with that came other issues of managing threads and preventing breakdown of the program and creating safeties and checks. Since War Thunder doesn't have APIs or support external plugins, I had to define how to send messages to the game posing as a player, and do other functions autonomously, as well as create commands for moderators on Discord to watch the players and make sure they behave and step in if needed. This of course was only a small small part of the whole bot, there are many other features.

After about 13 version releases of the project I created v2.0.0, where I integrated a website to control features that would otherwise only be accessible by complicated discord commands. The website uses the bot to link user's discord accounts and derive from that their level of permissions and what features to display. I wanted to create a site with the strongest possible security, so I had to figure out how to tokenize logins and create an API to protect my database, verify token validity and age, and sanitize potential malicious inputs. Also using the tokens I was able to sign them to users and decode them in my API to minimize database querying and streamline usage, preventing the possible replication of tokens or decoding.

Now the project is version 2.2.3, refactoring code for readability, efficiency, disk size, and better modularity for adding new code.

Statistics

The majority of the project excluding packages and XAMPP is java, having 76 classes and interfaces. The project uses about 7% of my CPU and 60MB memory at peak usage. It takes up about 707MB disk size, a notable improvement over the previous version's 881MB.
