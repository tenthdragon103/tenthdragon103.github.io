About Me in CS

I was first introduced to computers and programming at 7, when I wrote my first HTML and Scratch code. Around 8 I experimented with a Java learning course, which became my favorite language. I used my experience in my elementary school robotics club, where I programmed our bot to respond to our user inputs.

In high school I became more interested in creating larger and more complex programs than I had before, teaching myself concepts only taught in calculus 3 or higher, and learning formulas for my attempt at creating a 3D animation engine. Needless to say the project failed because I tried doing graphics in Java with AWT. However, it only made me want to keep going, so I kept creating new things with different styles and languages.
